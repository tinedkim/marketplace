Implementing EmailJS in React

Nick Boober
Feb 5 · 4 min read

EmailJS is a free platform that allows sending emails directly with JavaScript, with no back-end development. The developer creates one or more email templates and then triggers an email using the JavaScript SDK, specifying the template, and the dynamic parameters for rendering the email.

Registering and Installing EmailJS

Once the user registers at EmailJS you are provided a unique User ID and Access Token that you will need when setting your project up.

For free, the user gets a quota of 200 emails per month.

When using React, the user can either install EmailJS with the npm command below.

$ npm install emailjs-com — save

Bower,

$ bower install emailjs-com — save

Or, you can manually embed the EmailJS CDN directly into your index.html file as seen below.

Setting Up Your EmailJS Profile

The first thing you want to do after creating an account is provide an email service. This is the service that you want the emails to be managed by.

This can be found in your dashboard.

Once you select to add a new service you have to select a provider from the options below.

Once you select your provider you can change the Name and ID of the service provider. This ID will be used later on when creating your form in React.

Finally, once you connect the email that you want your emails sent to it will show up in your Email Services dashboard. You can always Edit, Test, and Delete this service.

Finally, you want to build your email template. EmailJS by default gives you a basic email template. These templates can be created in your dashboard as well.

Building your templates are very easy. Any variables you pass in must be wrapped in {{}}. Otherwise the information will not be passed correctly.

Other than that you can format your email however you want.

You can choose to send auto-reply emails. For my React form, I want the user to pass in their email so not only do I know who to respond to but I will also need to know where to send the auto-response.

You can also add attachments, contacts, etc.

From here you can test your emails automatically to make sure the formatting looks the way you want them to. I believe these test emails count against your quota for the month though. So use it sparingly.

Once done make sure to save your template.
